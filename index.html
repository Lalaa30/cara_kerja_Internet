Cara Kerja Internet Secara Umum

Internet bekerja melalui jaringan global komputer yang saling terhubung. Ketika Anda mengirim atau menerima data (seperti email atau halaman web), informasi tersebut dipecah menjadi paket-paket kecil. Paket-paket ini kemudian dikirim melalui berbagai jalur di internet dan akhirnya disusun kembali di tujuan.

HTTP dan HTTPS

HTTP (Hypertext Transfer Protocol) adalah protokol yang digunakan untuk mentransfer data di World Wide Web. HTTPS adalah versi aman dari HTTP, yang mengenkripsi data yang dikirim antara browser dan server web.

Cara kerja HTTP:

protokol HTTP menggunakan bentuk mode operasional permintaan dan respons. Ketika klien ingin mengambil informasi, ia menggunakan permintaan http ke server seperti yang ditunjukkan pada tahap-tahap berikut. Permintaan diterima oleh server dan dalam bentuk respons HTTP server mengembalikan data yang diminta klien atau pesan kesalahan. Ini terjadi melalui internet menggunakan port 80 secara default, untuk membantu identifikasi protokol ini sering disebut sebagai http atau protokol hip.

Cara kerja HTTPS:

HTTPS dapat dikatakan mirip dengan HTTP, hanya saja HTTPS juga menyediakan tingkat keamanan. HTTPS pertama-tama menciptakan koneksi antara klien dan server melalui SSL/TLS, yang meningkatkan keamanan dengan mengenkripsi komunikasi antara klien dan server. Ketika klien membuat permintaan untuk suatu sumber daya menggunakan https, maka server dan klien menyepakati kunci enkripsi yang akan digunakan untuk mengenkripsi data yang akan dikirimkan dalam sesi tertentu. Hal ini memastikan bahwa data yang dipertukarkan di antara mereka dienkripsi dan dikodekan sehingga tidak dapat dicegat.

Nama Domain

Nama domain adalah bagian penting dari URL situs web. Cara kerjanya seperti alamat situs web di Internet, sama halnya dengan alamat rumah fisik Anda di dunia nyata.

Cara kerja Nama Domain:

Saat pengguna memasukkan nama domain di browser, browser akan terhubung ke Domain Name System (DNS) yang merupakan jaringan global server atas nama domain yang berlokasi di seluruh dunia. DNS seperti database yang menyimpan nama domain dan alamat IP yang sesuai. Nama domain yang diminta kemudian dicocokkan dengan data pada DNS, dan akan diterjemahkan ke dalam alamat IP. Browser kemudian membuat koneksi dengan alamat IP yang dihasilkan, yang kemudian akan meneruskan permintaan browser awal ke server hosting situs web yang dimaksud. Server situs web kemudian akan menanggapi permintaan browser dan mengembalikan data ke browser untuk memuat situs web untuk pengguna.

Hosting

Hosting adalah layanan yang menyimpan file website Anda di server yang terhubung ke internet, memungkinkan website Anda diakses oleh orang lain.

Cara kerja Nama Domain:

  1. File Website disimpan di Server Hosting

    File website seperti gambar, video, script, database dan konten lainnya akan disimpan di server hosting.

  2. Pengunjung Akses Website

    Pengunjung akan mengetikkan alamat domain di browser, dan dari permintaan tersebut akan dikirimkan ke server hosting.

  3. Permintaan dikirimkan Ke Server Hosting

    Permintaan dari pengunjung akan dikirim ke server hosting yang akan memproses permintaan tersebut.

  4. Server Hosting Mengirimkan File Ke Browser

    Setelah permintaan diterima, server hosting akan mengirimkan file yang diminta seperti file HTML, gambar, atau konten lainnya ke browser pengunjung tersebut.

  5. Pengunjung Dapat Mengakses Website

    Setelah browser menerima file website dari server hosting, pengunjung dapat mengakses website dengan melihat halaman web yang terbentuk di browser.

DNS (Domain Name System)

Sistem Nama Domain (DNS) merupakan komponen penting infrastruktur Internet yang berperan penting dalam menghubungkan pengguna ke situs web, layanan, dan sumber daya di seluruh World Wide Web. Sistem ini pada dasarnya adalah "buku telepon" internet, yang menerjemahkan nama domain yang mudah digunakan (seperti www.example.com) menjadi alamat IP numerik (seperti 192.0.2.1) yang digunakan komputer dan perangkat jaringan untuk menemukan satu sama lain di internet.

Cara kerja DNS:

proses resolusi DNS melibatkan beberapa langkah, dengan permintaan yang berlanjut dari komputer pengguna ke server DNS rekursif ISP, melalui server DNS root dan TLD, dan akhirnya ke server DNS otoritatif untuk domain tertentu, sebelum mengembalikan alamat IP ke komputer pengguna untuk komunikasi lebih lanjut. Proses ini memastikan bahwa pengguna dapat mengakses situs web dan layanan menggunakan nama domain yang dapat dibaca manusia.

Browser

Browser adalah aplikasi yang digunakan untuk mengakses dan menampilkan halaman web yang berisi teks, gambar, video, atau suara yang ada di website maupun mesin pencari. Browser menerjemahkan kode HTML, CSS, dan JavaScript menjadi halaman web yang dapat dibaca.

Cara kerja Web Browser

Secara sederhana, urutan cara kerja web browser adalah sebagai berikut:

  1. Pengguna mengakses website di browser

    Anda sebagai pengguna bisa menginputkan alamat website, misalnya niagahoster.co.id, di address bar browser.

  2. Browser menerima permintaan akses pengguna

    Request akses Anda terhadap halaman niagahoster.co.id diterima browser. Nah, browser kemudian mengambil data berdasarkan permintaan tersebut ke DNS server.

  3. Browser mendapatkan IP website di DNS server

    Di DNS server, website memperoleh informasi berupa IP dari website niagahoster.co.id yang tadi diminta oleh Anda.

  4. Browser mengakses web server milik website

    Setelah memperoleh IP website, browser akan mengakses web server dari niagahoster.co.id untuk mendapatkan isi kontennya.

  5. Server mengirimkan isi halaman website

    Server hosting memberikan konten halaman niagahoster.co.id ke browser. Namun di sini, halaman web masih berupa script pemrograman.

  6. Browser menerjemahkan script halaman website

    Browser menerjemahkan kumpulan script pemrograman dari halaman niagahoster.co.id tersebut menjadi tampilan yang bisa dinikmati pengguna.

  7. Browser mengirimkan konten website ke pengguna

    Hasil terjemahan halaman niagahoster.co.id akan dikirimkan ke pengguna. Di sini, Anda bisa menikmati halaman tersebut dalam bentuk konten yang utuh.

  8. Browser menyimpan isi halaman berbentuk web cache

    Web cache adalah penyimpanan sementara untuk data website, dalam hal ini halaman niagahoster.co.id, yang bertujuan mempercepat akses Anda ke halaman tersebut di lain waktu.

  9. Browser menyimpan beberapa informasi berupa cookies

    Cookies adalah kumpulan informasi atau rekam jejak Anda ketika menjelajahi website, misalnya seperti informasi login, pengaturan bahasa, dan lain-lain.
